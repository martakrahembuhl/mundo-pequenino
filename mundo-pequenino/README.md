# Mundo Pequenino - Escola de Educação Infantil

## Sobre
Mundo Pequenino é uma escola de educação infantil dedicada ao desenvolvimento integral das crianças através de educação lúdica e respeitosa.

## Missão
Desenvolver integralmente crianças através de educação lúdica e respeitosa.

## Valores
- 🎨 Criatividade e Imaginação
- 🤝 Respeito e Inclusão
- 🌱 Desenvolvimento Integral
- 📚 Aprendizado Significativo

## Tecnologias
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto
- **public/** - Arquivos públicos (CSS, JS, imagens)
- **config/** - Configurações e conexão com banco de dados
- **includes/** - Componentes reutilizáveis
- **admin/** - Painel administrativo

## Como Instalar
1. Clone o repositório
2. Configure o banco de dados em config/database.php
3. Importe o arquivo banco-dados.sql no MySQL
4. Acesse http://localhost/mundo-pequenino
